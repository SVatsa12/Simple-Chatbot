# AI Task Assistant Chatbot

An intelligent task management assistant powered by Gemini AI.

## Features
- Natural language task management
- AI-powered chat interface
- Local storage for tasks (no database required)
- Clean React frontend with Vite
- Express backend as Gemini API proxy

## Project Structure
- `client/` - React frontend
- `server/` - Node.js/Express backend

## Getting Started

### Client Setup
```bash
cd client
npm install
npm run dev
```

### Server Setup
```bash
cd server
npm install
npm start
```
